# CS2 MOBILE

Este projeto tem como objetivo disponibilizar uma aplicação para visualização, busca e detalhamento de diversos itens relacionados ao universo de Counter Strike, incluindo **skins**, **stickers**, **highlights**, **crates** e **agents**.

##  Demonstração: 
![cs2 demo](/assets/cs2_demo.gif)
---



A seguir estão listadas todas as funcionalidades implementadas ao longo das entregas.

---

###  Skins
-  Listar todas as skins  
-  Filtrar / buscar skins por nome  
-  Mostrar detalhes de uma skin

  ---  

###  Stickers
-  Listar todos os stickers  
-  Filtrar / buscar stickers  
-  Mostrar detalhes de um sticker  

---

###  Highlights
-  Listar highlights  
-  Filtrar / buscar highlights  
-  Mostrar detalhes de um highlight, incluindo o vídeo

 ---

###  Crates
-  Listar crates  
-  Filtrar / buscar crates  

---

###  Crates 
-  Mostrar detalhes das crates  

---

###  Agents
-  Listar agents  
-  Filtrar / buscar agents  
-  Mostrar detalhes de um agent  

---

1. Clone o repositório:
   ```bash
   git clone https://github.com/tzydit/cs2Mobile

-> Aguarde o Android Studio sincronizar o Gradle.

-> Conecte um dispositivo físico ou inicie um emulador Android.

-> Clique em Run para executar o app.
